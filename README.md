# LogcatViewer library

### Feature:

- Priority filter
- Clear logcat
- Export as file 
- Floating window

### Integrate guide

1. Clone this library as a project module, add module dependence.

2. Add launch code in your code:

    ```java
    public void launchLogcatViewer() {
       LogcatActivity.launch(getContext());
    }
    ```
